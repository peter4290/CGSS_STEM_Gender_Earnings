# CGSS_STEM_Gender_Earnings
Empirical analysis of gender disparities in earnings among STEM workers using CGSS2021 data, with reproducible code and dataset.
数据来源说明
本研究使用的数据为中国综合社会调查（CGSS）2021年调查数据，由中国人民大学中国调查与数据中心（NSRC）于2023年3月31日发布。该调查覆盖中国大陆31个省（自治区、直辖市），有效样本量8148份，采用多阶段分层抽样方法。本研究严格遵循CGSS数据使用协议，仅用于学术研究，数据知识产权归发布机构所有，作者对数据使用合法性及研究结论独立负责。原始数据可通过CGSS官方网站获取（http://cgss.ruc.edu.cn）。
Data Source Description

The data used in this study are from the Chinese General Social Survey (CGSS) 2021, released on March 31, 2023 by the National Survey Research Center (NSRC) at Renmin University of China. The survey covers 31 provinces (autonomous regions and municipalities) in mainland China, with a valid sample size of 8,148 observations, and adopts a multi-stage stratified sampling design.

This study strictly complies with the CGSS data usage agreement and uses the data solely for academic research purposes. The intellectual property rights of the data belong to the issuing institution, and the author assumes full responsibility for the legality of data use and the conclusions drawn.

The original data can be accessed through the official CGSS website:
http://cgss.ruc.edu.cn

The file cgss2021_clean.do contains steps for data cleaning and variable construction, and will export a clean data excel.

The file cgss2021_analysis.do contains analysis of the clean data excel.
